To run system:
1) Open two seperate sessions
2) In one session execute command: ./server
3) In another session execute command: ./client

If for some reason the executables do not run, to build them:
1) Go to (root directory)/calculator/cmd/server
2) Enter command: go build -o (root directory)/calculator/bin/server
3) Go to (root directory)/calculator/cmd/client
4) Enter command: go build -o (root directory)/calculator/bin/client
